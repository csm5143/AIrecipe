# AIRecipe 项目概述

## 项目简介

AIRecipe 是一款基于微信小程序的 AI 智能食谱应用，用户可以通过拍照识别食材、AI 生成食谱、管理收藏夹等功能。

## 功能模块

### 用户端 (微信小程序)
- 食材拍照识别 (AI)
- 智能食谱推荐
- 收藏夹管理
- 购物清单
- 儿童营养食谱
- 健身餐专区

### 后台管理 (Web 系统)
- 用户管理
- 食谱管理 (CRUD)
- 食材管理
- 收藏管理
- 用户反馈处理
- Banner/公告运营
- 系统设置
- 数据分析看板
- 操作日志

### 移动端 (跨端应用)
- 基于 Taro 的 React Native / H5 应用
